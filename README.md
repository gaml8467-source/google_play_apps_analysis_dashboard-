##  Project Overview
This project presents an interactive **Power BI dashboard** analyzing Google Play Store applications to uncover insights about app performance, user engagement, and market trends.

The analysis includes **data cleaning, feature engineering, DAX calculations, and interactive visualizations** to support data-driven decision making.

---

##  Dataset Information
- **Source:** Kaggle
- **Rows:** ~9,366
- **Columns:** 13

### Key Features
- App
- Category
- Rating
- Reviews
- Size
- Installs
- Type (Free/Paid)
- Price
- Content Rating
- Genres
- Last Updated
- Android Version

---

##  Data Cleaning
- Handled **missing values (NaN)**
- Split long app names for better readability
- Fixed data types (e.g., installs, price, reviews)
- Removed inconsistencies and formatted columns

---

##  Analysis Process
- Exploratory Data Analysis (EDA)
- KPI creation using DAX
- Data modeling in Power BI
- Interactive dashboard design

---

##  KPIs
- Total Apps
- Average Rating
- Total Reviews
- Total Installs
